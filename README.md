# Weblancee — Premium Web Developer Portfolio

Modern, responsive portfolio website with glassmorphism design, animated gradients, and a working contact form.

## Features
- Premium dark-theme design with animated gradient text
- Fully responsive (mobile-first, 4 breakpoints)
- Contact form connected to backend API
- Project showcase with video previews
- Scroll-reveal animations and stat counters

## Deployment
Deployed on [Netlify](https://netlify.com) as a static site.

The `_redirects` file proxies `/api/contact` requests to the Render backend.
